/app/frontend\nsudo supervisorctl start frontend\n\n# Or manually:\nyarn start\n```\n**Access:** http://localhost:3000\n\n### 2️⃣ Start the Dashboard System\n```bash\n# Terminal 2: Start the dashboard frontend  \ncd /app/frontend-dashboard\nyarn start\n\n# Terminal 3: Start the dashboard backend\ncd /app/backend-dashboard\n# First time only: Set up database\nyarn db:migrate\nyarn db:seed\n# Then start server\nyarn dev\n```\n**Access:** http://localhost:3001/login\n\n### 3️⃣ You're Ready! 🎉\n\n---\n\n## 🔗 How They're Connected\n\n### From Landing Page → Dashboard:\n- **Header**: Click \"Login\" button (top-right)\n- **Contact Section**: Click \"Access Dashboard\" button\n- **Both open:** http://localhost:3001/login in new tab\n\n### Navigation Flow:\n```\nVisitor lands on:\nhttp://localhost:3000 (Marketing Site)\n          ↓\nClicks \"Login\" or \"Access Dashboard\"\n          ↓\nRedirects to:\nhttp://localhost:3001/login (Authentication)\n          ↓\nAfter login success:\nhttp://localhost:3001/dashboard (User Portal)\n```\n\n---\n\n## 🔑 Demo Login Credentials\n\n### SuperAdmin (Full Access)\n- **Email:** admin@lumaa.ai\n- **Password:** admin123\n- **Can:**
